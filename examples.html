jQuery Data Attribute Search

This jQuery plugin allows you to search a page based on tags provided in a data attribute.

This plugin gets called on an input elemnt type of text, and here is the syntax.

$("element").dataAttrSearch(Data,Search,Clear,Status)

Data - Type: String
Pass the data attribute you'd like to search, can pass either the whole name 'data-tags' or just the name of the data attribute 'tags'
Defaults to: tags

Search - Type: Boolean
Determine weither or not you'd like the search button to be added to the element
Defaults to: false

Clear - Type: Boolean
Determine weither or not you'd like the clear button to be added to the element
Defaults to: false

Status - Type: Boolean
Determine weither or not you'd like the status of the search to be shown
Example: Search Reuturned 1 out of 13
Defaults to: false

Below you will find examples of options provided with the plugin.

Default Options

This will generate a search box with no buttons, that will by default search the data attribute 'tags'

$('input.none').dataAttrSearch()

Search Button Enabled

This will generate a search box with a search button, that will by default search the data attribute 'tags'

$('input.search').dataAttrSearch(null,false)

Clear Button Enabled

This will generate a search box with a clear button, that will by default search the data attribute 'tags'

$('input.search').dataAttrSearch(null,false,true)

All Options Enabled

This will generate a search box with a search button, a clear button, and will display the results status. Passed no paramters for the data attribute it will search 'tags' by default.

$('input.all').dataAttrSearch(null,true,true,true)

Custom Data Attribute

This will search the specified data attribute, rather than the default of tags.

$('input.all').dataAttrSearch('potato')
